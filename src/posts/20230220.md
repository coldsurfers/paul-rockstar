# Android 에뮬레이터에 Play Store 활성화하기

> AVD config.ini 파일 두 줄만 바꾸면 된다.

---

## config.ini 열기

```bash
vi ~/.android/avd
```

vim으로 AVD 디렉토리를 열면 `your_emulator_name.avd` 폴더가 보인다. 들어가서 `config.ini`를 연다.

---

## 수정할 항목 1 — PlayStore.enabled

```ini
# 변경 전
PlayStore.enabled = false

# 변경 후
PlayStore.enabled = true
```

---

## 수정할 항목 2 — image.sysdir.1

```ini
# 변경 전
image.sysdir.1 = system-images/android-32/google_apis/arm64-v8a/

# 변경 후
image.sysdir.1 = system-images/android-33/google_apis_playstore/arm64-v8a/
```

`google_apis` → `google_apis_playstore` 로 바꾸는 것이 핵심이다. Android 버전(32→33)도 playstore 이미지가 있는 버전으로 맞춰야 한다.

---

*에뮬레이터를 재시작하면 Play Store가 활성화된다.*
