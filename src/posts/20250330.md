# DRM의 구조 — FairPlay, Widevine, 그리고 AES-CTR

> 스트리밍 서비스에서 콘텐츠가 어떻게 보호되는가. 암호화부터 플랫폼별 구현까지.

---

## DRM이란

DRM(Digital Rights Management)은 디지털 콘텐츠의 불법 복제 및 무단 사용을 방지하는 기술이다. 영상 스트리밍, 전자책, 게임에서 주로 쓰인다.

핵심 기능 네 가지:
1. **콘텐츠 암호화** — AES-CTR
2. **라이선스 제어** — 인증된 사용자만 접근
3. **기기 인증** — 승인된 기기에서만 재생
4. **멀티 DRM** — Widevine(Google), PlayReady(Microsoft), FairPlay(Apple)

---

## 암호화 방식: AES-CTR

각 블록마다 증가하는 카운터 값을 암호화해 키 스트림을 생성하고 평문과 XOR 연산한다.

```
(Nonce + 1 → 암호화) ⊕ P1 → C1
(Nonce + 2 → 암호화) ⊕ P2 → C2
(Nonce + 3 → 암호화) ⊕ P3 → C3
```

랜덤 액세스가 가능해 스트리밍에 적합하고 병렬 처리도 된다. 단, Nonce 재사용 시 보안이 취약해진다.

## 라이선스 서버와 키 교환

콘텐츠를 재생하려면 라이선스 서버에서 키를 받아야 한다.

```
AES(대칭키)  → 콘텐츠 암호화
RSA(공개키)  → 대칭키 보호 및 전송
최종 복호화  → 대칭키(AES) 사용
```

흐름:
1. 사용자가 콘텐츠 재생 요청
2. 클라이언트가 DRM 라이선스 서버에 요청 전송
3. 서버에서 기기 및 계정 검증
4. 허가된 경우 암호화된 대칭키 제공 (공개키 암호화)
5. 클라이언트가 키를 복호화해 콘텐츠 해독

## 웹: FairPlay DRM (EME API)

```html
<video id="video" controls></video>
<script>
  navigator.requestMediaKeySystemAccess('com.apple.fps.1_0', [{
    initDataTypes: ['cenc'],
    videoCapabilities: [{ contentType: 'video/mp4; codecs="avc1.4d401e"' }]
  }])
  .then(access => access.createMediaKeys())
  .then(mediaKeys => {
    document.getElementById('video').setMediaKeys(mediaKeys)

    const session = mediaKeys.createSession()
    session.onmessage = (e) => {
      const xhr = new XMLHttpRequest()
      xhr.open('POST', 'https://license.example.com/fairplay')
      xhr.setRequestHeader('Content-Type', 'application/octet-stream')
      xhr.send(e.message)
    }
  })
</script>
```

## 웹: Widevine DRM (EME API)

```js
navigator.requestMediaKeySystemAccess('com.widevine.alpha', [{
  initDataTypes: ['cenc'],
  videoCapabilities: [{ contentType: 'video/mp4; codecs="avc1.42E01E"' }]
}])
.then(access => access.createMediaKeys())
.then(mediaKeys => {
  video.setMediaKeys(mediaKeys)

  return fetch('https://license.example.com/widevine', {
    method: 'POST',
    body: JSON.stringify({ type: 'license', initData: new Uint8Array() }),
    headers: { 'Content-Type': 'application/json' }
  })
  .then(res => res.arrayBuffer())
  .then(data => mediaKeys.setServerCertificate(new Uint8Array(data)))
})
```

## Android: ExoPlayer + Widevine

```kotlin
class DRMPlayerActivity : AppCompatActivity() {
    private lateinit var exoPlayer: ExoPlayer

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)

        val dataSourceFactory = DefaultHttpDataSource.Factory()
        val drmSessionManager = DefaultDrmSessionManager(
            DefaultDrmProvider("https://license.example.com/widevine")
        )

        val mediaItem = MediaItem.fromUri(Uri.parse("https://example.com/video.mpd"))
        val mediaSource = DashMediaSource.Factory(dataSourceFactory).createMediaSource(mediaItem)

        exoPlayer = ExoPlayer.Builder(this).build()
        exoPlayer.setMediaSource(mediaSource)
        exoPlayer.prepare()
        exoPlayer.play()
    }

    override fun onDestroy() {
        super.onDestroy()
        exoPlayer.release()
    }
}
```

## iOS: FairPlay + AVFoundation

```swift
class DRMPlayerViewController: UIViewController {
    var player: AVPlayer?

    override func viewDidLoad() {
        super.viewDidLoad()

        let asset = AVURLAsset(url: URL(string: "https://example.com/video.m3u8")!)
        asset.resourceLoader.setDelegate(self, queue: .main)

        player = AVPlayer(playerItem: AVPlayerItem(asset: asset))

        let layer = AVPlayerLayer(player: player)
        layer.frame = view.bounds
        view.layer.addSublayer(layer)
        player?.play()
    }
}

extension DRMPlayerViewController: AVAssetResourceLoaderDelegate {
    func resourceLoader(
        _ resourceLoader: AVAssetResourceLoader,
        shouldWaitForLoadingOfRequestedResource loadingRequest: AVAssetResourceLoadingRequest
    ) -> Bool {
        let request = URLRequest(url: URL(string: "https://license.example.com/fairplay")!)

        URLSession.shared.dataTask(with: request) { data, _, error in
            guard let data, error == nil else {
                loadingRequest.finishLoading(with: error)
                return
            }
            loadingRequest.dataRequest?.respond(with: data)
            loadingRequest.finishLoading()
        }.resume()

        return true
    }
}
```

## 정리

| | Widevine | FairPlay | PlayReady |
|---|---|---|---|
| 제공 | Google | Apple | Microsoft |
| 플랫폼 | Android, Chrome | iOS, Safari | Windows |
| 웹 API | EME | EME | EME |

*DRM은 AES-CTR로 콘텐츠를 암호화하고, RSA로 키를 보호해 전달한다. 플랫폼마다 구현 방식은 다르지만 구조는 같다.*
