# paul-rockstar

Personal portfolio site for Paul Choi — Frontend Engineer & Founder of [COLDSURF](https://coldsurf.io).

Live at [paul-rockstar.coldsurf.io](https://paul-rockstar.coldsurf.io)

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- Framer Motion

## Dev

```bash
pnpm install
pnpm dev
```
